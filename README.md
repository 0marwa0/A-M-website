# M2A Ai — Futuristic Next.js Experience

Dark neon, glassmorphic Next.js 14 (App Router) + Tailwind build for the M2A Ai brand.

## Setup
1) Install deps: `npm install`
2) Run dev server: `npm run dev` then open the shown localhost URL.
3) Lint: `npm run lint`

## Notes
- App Router in `app/` with reusable UI: neon button, glass cards, section shell, particle layer.
- Global styling lives in `app/globals.css` with gradients, glows, and animations tuned to the reference theme.
